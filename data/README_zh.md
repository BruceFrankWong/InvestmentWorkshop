# 目录介绍

本目录下的文件为测试用数据。这些 <.csv> 文件是真实的交易数据，通过 [天勤SDK](https://www.shinnytech.com/tianqin/) 下载。

<.zip> 文件（包括 <.z01>、<.z02> ... 等等）是考虑节约容量便于上传到 git。使用前记得解压缩并放在本目录下。

文件命名格式：

```
股票：
<交易所代码>.<品种代码>_<周期>.csv

期货：
<交易所代码>.<品种代码><交割年月>_<周期>.csv

期权：
<交易所代码>.<品种代码><交割年月><方向><行权价>_<周期>.csv
```

其中：

- <交易所代码>:
  - SSE

    上海证券交易所

  - SZSE

    深圳证券交易所
  
  - CZCE

    郑州商品交易所
  
  - DCE

    大连商品交易所
  
  - SHFE

    上海期货交易所
  
  - CFFEX

    中国金融期货交易所

  - HKEX
  
    香港证券交易所
  
- <品种代码>：

    详见各交易所，不一一详述。需要注意大小写。

- <交割年月>

    四位数字，以年份的后两位 + 两位的月份（不足两位的前面补0）。

- <周期>

    Tick、Minute、Day。
